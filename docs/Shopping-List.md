You can buy almost everything on Amazon, but it will be more expensive. I highly recommend going through AliExpress. It will take a bit longer, but it is significantly cheaper, and you will have to source the modem from there anyway. Buying a Raspberry Pi should easily be possible locally, there is an [official reseller list](https://www.raspberrypi.com/resellers).
